# AI自動化ラボ

AIを使ってホームページ、掲示板サイト、SNS投稿管理、Web自動化ツールを作っていくためのホームページです。

## ファイル

- `index.html`: ページ本体
- `styles.css`: 見た目
- `script.js`: 年号などの小さな動き

## ローカルで見る

`index.html` をブラウザで開けば見られます。

## 公開

このサイトはGitHubにアップロードして、Vercelで公開します。

公開URL:

```text
https://ai-web-automation-lab.vercel.app
```

## あとで差し替えるところ

- メール: `hello@example.com`
- Xリンク: `https://x.com/`
- 文章や実績
